## Notes for Localization

When adding translations for other languages:
1. Keep button/control terms consistent with existing UI strings
2. Adjust list formatting (bullets) if needed for RTL languages
3. Feature tier limits (numbers) should stay consistent across languages
4. Technical terms like "EQ", "fade-out" may not need translation
