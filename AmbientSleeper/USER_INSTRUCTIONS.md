## Getting Started

Ambient Sleeper is organized into three main tabs. Each tab provides different functionality to help you create and manage your perfect audio experience.

### Quick Start Guide
1. Go to the **Library** tab
2. Tap sounds to add them to your mix
3. Switch to the **Playback** tab
4. Adjust individual volumes
5. Press "Start Mix" to begin playback
6. Optionally set a timer in the **Timer** tab

---

## Library Tab

**Browse and organize your audio content**

### Sound Bundles
- Collections of related ambient sounds (nature, white noise, industrial, etc.)
- Tap any sound to add it to your current mix or playlist
- All sounds are available in all tiers

###  Saved Playlists  
- View and manage your saved playlists
- Load a playlist to queue multiple sounds for sequential playback
- Requires **Standard tier or higher**

### Mix Playlists
- Advanced feature for creating scheduled transitions between different mixes
- Set custom durations for each mix
- Configure crossfade transitions
- Requires **Premium tier or higher**

## Playback Tab

**Control your audio playback**

### Mix Mode
- **Overlay multiple sounds simultaneously**
- Adjust individual volume levels for each sound using sliders
- Save your favorite combinations with custom names
- Free tier: 2 sounds | Standard: 3 sounds | Premium: 10 sounds | Pro+: 20 sounds

**How to create a mix:**
1. Add sounds from the Library tab
2. Adjust each sound's volume in Playback tab
3. Tap "Save Mix" to save your configuration
4. Give it a descriptive name (e.g., "Bedtime", "Focus", "Rainy Day")

### Playlist Mode (Standard+)
- **Play sounds sequentially, one after another**
- Enable loop mode to repeat the playlist continuously
- Add sounds from Library to your playlist queue
- Save playlists for future use

**How to create a playlist:**
1. Switch to Playlist mode in Playback tab
2. Add sounds from Library tab
3. Arrange order as desired
4. Enable loop if you want continuous playback
5. Tap "Save Playlist" to save

### Mix Playlist Mode (Premium+)
- **Create scheduled rotations of different saved mixes**
- Each mix can have a custom duration
- Configure crossfade transitions between mixes
- Perfect for evolving soundscapes throughout your session

**How to create a mix playlist:**
1. First, create and save individual mixes
2. Switch to Mix Playlist mode in Playback tab
3. Add your saved mixes
4. Set duration for each mix (default: 3 minutes)
5. Set transition/fade time if enabled
6. Enable loop for continuous rotation

### Playback Controls
- **Start/Stop**: Begin or end playback (color-coded: green for play, red for stop)
- **Volume Sliders**: Adjust individual sound volumes in mix mode
  - Shows "Volume: 85%" label above each slider
  - Real-time percentage display
- **Loop Toggle**: Repeat playlists continuously
- **EQ Button**: Access equalizer settings (Premium+)
  - Disabled with explanation if not available in your tier
- **Export/Import**: Back up or share your configurations (Standard+)
- **Audio Settings**: Configure fade-out and alarm integration

### Visual Feedback
- **Active tab**: Highlighted with dark background and white text
- **Disabled tabs**: Shown at 50% opacity with lock icon when premium features are locked
- **Empty states**: Helpful messages when you haven't added any sounds yet
- **Item counts**: "Sounds in mix: 2 of 10" shows current usage vs tier limit
- **Tier restrictions**: Orange warning boxes clearly explain locked features

## Timer Tab

**Set playback timers for automatic stop**

The Timer tab helps you automatically stop audio playback after a set duration or at a specific time. The interface uses visual highlighting to show which mode is active.

### Timer Modes

**Duration Mode (Countdown Timer)**
- Set how long you want playback to continue (e.g., 30 minutes, 1 hour)
- Perfect for naps, meditation sessions, or timed focus periods
- Audio will fade out gradually when the timer expires
- The active mode is highlighted with a green border for easy identification
- **Free tier: 15-minute session limit | Paid tiers: unlimited duration**

**How to use:**
1. Select "Stop after duration" (highlighted in green when selected)
2. Use the time picker to set hours and minutes
3. Tap "? Start Timer" to begin
4. Watch the countdown in the Timer Status section

**Stop At Time Mode (Scheduled Stop)**
- Set a specific clock time when audio should stop (e.g., 11:00 PM, 7:30 AM)
- Perfect for bedtime routines or wake-up schedules
- Audio fades out at exactly the specified time
- The active mode is highlighted with a green border

**How to use:**
1. Select "Stop at specific time" (highlighted in green when selected)
2. Use the time picker to choose your stop time
3. Tap "? Start Timer"
4. Check the Timer Status to see time remaining

### Timer Status
The status box (highlighted in blue) shows:
- **Time Remaining** in hours, minutes, and seconds (hh:mm:ss format)
- Clear explanation of what will happen when timer expires
- Real-time countdown updates

### Timer Controls
Organized for easy access with color-coded buttons:

**Primary Controls:**
- **? Start Timer** (Green) - Begin the countdown or scheduled timer
- **? Cancel Timer** (Red) - Stop and reset the timer

**Quick Adjustments:**
- **? Add 5 min** - Extend the timer by 5 minutes without starting over
- **? Remove 5 min** - Reduce the timer by 5 minutes
- Perfect for making quick tweaks while timer is running

**Alarm Control:**
- **?? Stop Alarm** (Orange) - Stop the alarm if it's currently playing

### Alarm Integration (Standard+)
- Optionally play a system alarm sound when timer expires
- Choose your preferred alarm sound from device sounds
- Alarm plays after audio fades out
- Helps ensure you wake up or notice when the timer ends
- **Requires Standard, Premium, or Pro+ tier**

**Free tier note:** Uses your device's default alarm sound only

### Fade-Out Control
Set in the Audio Settings (toolbar button):
- Customize how long the fade-out takes when timer expires
- Smooth transitions prevent jarring stops
- **Tier limits:**
  - Free: Up to 3 seconds
  - Standard: Up to 15 seconds  
  - Premium: Up to 60 seconds
  - Pro+: Up to 120 seconds

### Timer Tips

**?? The Timer tab includes helpful tips:**
- **Duration mode**: Best for naps (e.g., 30 minutes)
- **Stop At Time mode**: Best for bedtime (e.g., 11:00 PM)
- Audio fades out gradually when timer expires
- Use +/- 5 min buttons to quickly adjust a running timer

**Quick Reference:**
- 15-30 minutes: Quick naps, meditation

## Advanced Features

### Creating Mixes
Mixes allow you to overlay multiple ambient sounds simultaneously for a rich soundscape.

**Best Practices:**
- Start with 1-2 sounds and gradually add more
- Use one sound as a "base" (higher volume) and others for texture (lower volume)
- Mix complementary sounds (e.g., rain + distant thunder + wind)
- Save different mixes for different activities

**Volume Mixing Tips:**
- Base sound: 80-100% volume
- Accent sounds: 30-60% volume
- Subtle textures: 10-30% volume

### Creating Playlists (Standard+)
Playlists play sounds sequentially, perfect for guided relaxation or variety.

**Use Cases:**
- Meditation sessions with changing ambiences
- Sleep routines that evolve (active ? calming ? silence)
- Study sessions with timed breaks
- Yoga or exercise routines

### Mix Playlists (Premium+)
Advanced feature combining mixes with scheduled transitions.

**Example Scenarios:**
- **Sleep Routine**: Forest (10 min) ? Ocean (15 min) ? Light Rain (30 min)  
- **Focus Session**: White Noise (25 min) ? Nature (5 min break) ? Repeat
- **Meditation**: Tibetan Bowls (5 min) ? Wind Chimes (10 min) ? Silence

**Configuration:**
- Duration: How long each mix plays
- Transition: Crossfade time between mixes
- Loop: Repeat the entire sequence

### Equalizer (EQ) (Premium+)
Customize the frequency balance of your audio for the perfect sound.

**Locked Feature Notification:**
If you're on Free or Standard tier, you'll see an orange warning box explaining:
- What the equalizer does
- Which tiers include it (Premium for 5-band, Pro+ for 10-band)
- How to upgrade

**Premium Tier: 5-Band Graphic EQ** ??
- Adjust 5 frequency bands with visual sliders
- Each band shows frequency (Hz) and gain (dB)
- Min/max labels on sliders (-12 dB to +12 dB)
- Visual cards for each band with color-coded gain display
- **Frequency bands:**
  - Bass (60-250 Hz)
  - Low Mids (500 Hz - 1 kHz)
  - Mids (1-2 kHz)
  - High Mids (4 kHz)
  - Treble (8-16 kHz)

**Controls:**
- **? Apply** (Green button) - Apply your EQ changes
- **? Reset** - Revert to previous settings
- **0 Flat** - Reset all bands to 0 dB (no EQ)

**Pro+ Tier: 10-Band Parametric EQ** ??
- Advanced control with 10 frequency bands
- Each band has:
  - **Gain** (-12 to +12 dB) - Boost or reduce frequency
  - **Q Factor** (0.3 to 10) - Controls bandwidth (width of affected frequencies)
    - Lower Q = wider range affected
    - Higher Q = narrow, precise control
- Visual sliders with min/max labels
- Color-coded displays (gain=green, Q=blue)

**Quick Presets:**
- Highlighted preset section with purple border
- Dropdown menu with preset choices
- **Available presets:**
  - Flat (no EQ)
  - Bass Boost
  - Vocal Enhance
  - Treble Bright
  - And more
- **?? Load** button to apply preset

**EQ Tips (Built-in):**
The EQ page includes helpful tips:
- **Bass (60-250 Hz)**: Boost for deeper, richer sound
- **Mids (500-2k Hz)**: Adjust for clarity and warmth
- **Treble (4k-16k Hz)**: Enhance for brightness and detail
- **Start with small adjustments** (±3 dB)
- **Pro+ Advanced Tips**:
  - Q Factor: Lower = wider frequency range affected
  - Use presets as a starting point, then fine-tune
  - Avoid boosting too many frequencies (causes distortion)
  - Make subtle adjustments for best results

**How to Use EQ:**
1. Open EQ from Playback toolbar (top right)
2. Adjust sliders to taste
3. See real-time gain values
4. Pro+ users: Load a preset first, then customize
5. Tap **? Apply** to save changes
6. Tap **? Reset** if you don't like it
7. Tap **0 Flat** to start over

**Visual Enhancements:**
- Each band is in a card with border and background
- Gain values shown in green, bold text
- Sliders have color-coded tracks (green when filled)
- Pro+ Q values shown in blue text
- Preset section highlighted for easy access
