🔐Our Privacy Principles

✅ Core Commitments:
  • Privacy First: Your data belongs to you, not us
  • Local Storage: Data stays on your device whenever possible
  • Encryption: Sensitive data is always encrypted
  • No Tracking: We don't track your behavior or location
  • No Selling: We never sell or share your data with third parties
  • Minimal Collection: We only collect what's essential for app functionality
  • Transparency: Clear disclosure of all data practices

📊Data Collection and Usage

1. Local App Data (Stored on Device)

Most Celestryll apps store data locally on your device only:

BadlyDefined Specific Data:

  • Game Progress: Puzzles completed, current streak, total points, difficulty preferences
  • Statistics: Number of puzzles solved, average attempts, best performance, hint tokens
  • Optional Email: If provided, used only for account recovery and notifications

General App Data (may apply to other Celestryll apps):

  • App preferences and settings
  • User-created content (notes, saves, etc.)
  • Progress and achievements
  • Usage statistics (local only, not transmitted)

Storage Method:

  • ✅ All sensitive data encrypted using device-specific keys
  • ✅ Encryption keys stored in platform secure storage (Android Keystore, iOS Keychain)
  • ✅ Database files protected from unauthorized access
  • ✅ No automatic cloud backup of app-specific data

2. Data We DO NOT Collect

❌ Celestryll Apps DO NOT Collect:
  • Location data (GPS, network-based location)
  • Personal identifying information (beyond optional email when explicitly provided)
  • Contacts or address book
  • Photos or media files (unless app feature explicitly requires it)
  • Device identifiers for tracking (IMEI, UDID, advertising ID)
  • Browsing history or activity in other apps
  • Microphone or camera data (unless app feature requires it)
  • Calendar or call logs
  • Analytics or behavioral tracking

3. Optional Features & Permissions

Some features may request permissions. These are always:

  • Optional: You can decline and still use core app features
  • Purpose-Specific: Only used for the stated feature
  • Revocable: You can revoke at any time in system settings
Permission Purpose Apps
Network State Check connectivity status All apps

🤝Data Sharing and Transfers

Celestryll does not share, sell, or transmit your personal data to third parties.

Limited Data Sharing (User-Initiated Only):

  • Share Feature: When you use "Share" buttons, only the text/content you choose to share is transmitted (no personal data)
  • Email Features: Only if you explicitly provide email and choose to use email features
  • Cloud Backup: If you opt-in to cloud backup (future feature), data is encrypted before upload

Third-Party Services Currently Used:

  • SQLite: Local database (no network access)
  • .NET MAUI: Microsoft framework (telemetry disabled)
  • CommunityToolkit.Maui: Open source UI components (no telemetry)

NO Analytics, Advertising, or Tracking SDKs are integrated.

🔒Data Security

Security Measures:

  • ✅ Encryption at Rest: All sensitive data encrypted using AES-256 or equivalent
  • ✅ Secure Key Storage: Encryption keys stored in platform secure storage (Android Keystore, iOS Keychain)
  • ✅ Code Obfuscation: Release builds use R8/ProGuard to protect code
  • ✅ Minimal Permissions: Apps request only essential permissions
  • ✅ Regular Updates: Security patches and updates provided regularly
  • ✅ No Plain Text Storage: Sensitive data never stored in plain text

⚖️Your Rights

You have the following rights regarding your data:

Right How to Exercise
Access View your data in app settings/profile section
Delete Uninstall app (removes all local data), or use in-app delete option
Modify Edit data directly in app settings
Export Use app's export feature (if available) or contact support
Opt-Out Decline optional features, revoke permissions in system settings

🕐Data Retention

  • Local Data: Retained until you delete it or uninstall the app
  • App Uninstall: Permanently deletes all locally stored data
  • Cloud Data: (If applicable) Retained until you request deletion
  • No Server Backups: We do not maintain server backups of your local data

👶Children's Privacy

Celestryll apps are designed to be safe for all ages:

  • ✅ No data collection from children under 13 (COPPA compliant)
  • ✅ No social features or user-to-user communication
  • ✅ No external links (except privacy policy and support)
  • ✅ No advertising targeting children
  • ✅ No in-app purchases without parental consent mechanisms

Parent/Guardian Rights: If you believe a child under 13 has provided personal information, contact us immediately at privacy@celestryll.com and we will delete it.

🌍International Data Transfers

Since Celestryll apps primarily store data locally on your device:

  • No routine international data transfers occur
  • Data stays in the country where your device is located
  • If cloud features are added, data will be encrypted and stored in compliant data centers
