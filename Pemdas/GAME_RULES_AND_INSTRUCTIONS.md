# ?? PEMDAS - Complete Game Rules & Instructions

## ?? Game Overview

PEMDAS is a daily math puzzle game with two distinct game modes: **Solve It** and **Build It**. Each day presents a new challenge that tests your mathematical reasoning and problem-solving skills.

---

## ?? Game Modes

### **Solve It Mode**
Find the missing values in an equation to make it true.

#### Types:
- **Easy**: Find 1 missing value (represented by `?`)
- **Hard**: Find 2 missing values (represented by `A` and `B`)
- **Tricky**: Find 3 missing values (represented by `A`, `B`, and `C`)
- **Boss**: Find 2 missing values with advanced operations (represented by `X` and `Y`)

---

### **Build It Mode**
Use the given digits and operators to create an equation that equals the target number.

#### Types:
- **Medium**: Use 4 specific digits
- **Creative**: Use 4 specific digits (rewards elegant solutions)
- **Speed**: Use all digits 1-9 within 60 seconds

---

## ?? **CRITICAL RULES**

### **Rule 1: Integer Solutions Only** ??
- ? All answers must be **whole numbers** (integers)
- ? No fractions or decimals allowed
- ? Example: Answer = 5 (correct)
- ? Example: Answer = 2.5 (incorrect)

### **Rule 2: Different Variable Values** ??
**For puzzles with multiple variables (A, B, C or X, Y):**

- ? **All variables MUST have DIFFERENT values**
- ? **Two or more variables cannot have the same value**

**Examples:**

? **CORRECT:**
```
(A + B) × C = 30
A = 3, B = 2, C = 6  ? All different!
A = 6, B = 4, C = 3  ? All different!
A = 5, B = 5, C = 3  ? A and B are same!
```

? **CORRECT:**
```
(X˛ + 4) ÷ (Y - 1) = 4
X = 2, Y = 3  ? Different!
X = 2, Y = 2  ? Same values!
```

? **CORRECT:**
```
(A × 2) + (B ÷ 2) = 14
A = 5, B = 8  ? Different!
A = 5, B = 5  ? Same values!
```

### **Rule 3: Use All Given Digits (Build It Mode)** ??
- ? Must use **every** available digit exactly once
- ? Example: Given 1,2,3,4 ? Must use all four
- ? Cannot skip digits: `1+2+3` (missing 4)
- ? Cannot repeat digits: `1+1+2` (1 used twice)

### **Rule 4: Operator Rules**
- ? Can use: `+` (add), `-` (subtract), `×` (multiply), `÷` (divide)
- ? Can use parentheses `( )` for order of operations
- ?? Exponents (`˛`, `ł`) only in Boss puzzles
- ?? Follow standard order of operations (PEMDAS/BODMAS)

## ?? How to Play

### **Solve It Mode:**

1. **Look at the equation** - Identify the missing values (?, A, B, C, or X, Y)
2. **Use labeled input boxes**:
   - Easy: Single entry field for `?`
   - Hard/Boss: Two boxes labeled `A =` and `B =` (or `X =` and `Y =`)
   - Tricky: Three boxes labeled `A =`, `B =`, `C =`
3. **Enter your answers**:
   - Type integers into each box
   - Remember: All values must be DIFFERENT
4. **Tap Submit** to check your answer
5. **Get instant feedback**:
   - ? Green = Correct!
   - ? Red = Try again!

### **Build It Mode:**

1. **See the target number** - This is what you need to reach
2. **See available digits** - These are the only numbers you can use
3. **Use the calculator keypad**:
   - Tap digit buttons to add numbers
   - Tap operator buttons (+, -, ×, ÷)
   - Use parentheses ( ) for order of operations
   - Disabled buttons = digits not available
4. **Build your equation**:
   - Example: `(1 + 3) × 2 + 4`
   - Must use ALL available digits
5. **Tap Submit** to check
6. **Get instant feedback**:
   - ? Green = Correct!
   - ? Red = Try again!
